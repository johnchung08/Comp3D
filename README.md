# Compensation-in-3D, Comp3D

This folder contains MATLAB routines that demonstrate Comp3D

# System requirement
The codes have been tested on a standard desktop PC running Windows 11, MATLAB 2023a with the following toolboxes Signal Processing Toolbox (medfilt.m), Image Processing Toolbox, Statistics and Machine learning Toolbox, Scattered Data Interpolation and Approximation using Radial Base Functions
